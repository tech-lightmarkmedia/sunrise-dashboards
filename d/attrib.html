UTM Attribution — Guide

What This Dashboard Shows

Every contact's landing URL is stored by HubSpot with all query parameters intact. This dashboard parses those URLs to extract utm_source, utm_medium, and utm_campaign — giving cleaner attribution than HubSpot's DD1/DD2 fields, which lose information through inconsistent mapping.

Attribution Modes

First Touch: Parses the contact's first tracked landing URL. For contacts imported before HubSpot tracking (~March 2023), this represents their first HubSpot-tracked interaction, not their original acquisition source.

Last Touch Before Discovery: For contacts who had a discovery call in the date range, finds the most recent page visit or form submission URL before the discovery date. Shows which campaign was the final influence before booking. The table only shows contacts who reached discovery.

Last Touch Before Commitment: Same logic, but before the InvestNext commit-stage date. Only shows contacts who committed in the date range. Note: commitment tracking via the InvestNext commit pipeline stage (Fund V) began approximately December 2024. Earlier periods will show no commitment data.

Last Touch Before Investment: Same logic, but before the investment's funds_received date. Only shows contacts who funded in the date range.

Recent Activity: Parses the contact's most recent landing URL (hs_analytics_last_url). Shows what contacts are engaging with now, regardless of funnel stage.

The † Fallback

Rows with † had no UTM parameters set on the landing URL, or HubSpot did not capture them. The campaign column shows "(no URL campaign)" followed by the best available alternative: the first conversion event name (e.g. "Facebook Lead Ads: LM[FB]- Tax 2026"), the landing page path, or the HubSpot DD1/DD2 source data.

Campaign ID Resolution

Some ad URLs use the numeric campaign ID instead of a human-readable name for utm_campaign (e.g. "120247034873820439"). The dashboard resolves these to campaign names using data from Meta, Google Ads, and LinkedIn APIs. Resolved names show a green ID→ badge. Unresolvable IDs display as-is.

Filters

Use Source and Medium dropdowns to isolate specific platforms or channels. KPIs update with the filter. All/None buttons at the top for bulk selection.

Campaign Search: Type 3 or more characters to filter the table to rows where the campaign name contains your search term. Useful for finding specific campaigns across all sources.

Milestone Touch Logic

For "Last Touch Before Discovery/Commitment/Investment" modes, the dashboard finds the marketing touch that brought the contact to the conversion action — not the conversion action itself. It fetches recent page visits and form submissions before the milestone date, skips the conversion page (booking form, commitment page, etc.) and non-marketing URLs (HubSpot meeting links, InvestNext), and returns the first event with a different URL — preferring events with UTM parameters.

Funnel Columns

Discoveries, Commitments, and Investments show what each campaign's contacts produced. Click any number to see the records.

Data Source

Contacts fetched live (chunked by month). Meetings and investments from the nightly R2 cache. Commitments and milestone events fetched live.
